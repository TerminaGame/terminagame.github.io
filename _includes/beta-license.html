About the beta program

The Termina macOS Beta Program is a means of publicly beta-testing the macOS version of Termina, the SpriteKit GUI for the Termina project. The beta program works hand-in-hand with Visual Studio’s AppCenter program as a means of testing and distribution.

Privacy Policy

Because Termina for macOS (Beta) works with Visual Studio’s AppCenter, data is sent to the cloud. Here’s a list of what gets sent and what doesn’t:

The following data is not get given to us:

  • Player data or settings.json files

The following data is given to us:

  • File-loading or file-saving errors
  • Sprite interaction errors
  • Backend errors that don’t collect player data
